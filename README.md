# Astera

This is a [Next.js](https://nextjs.org/) project bootstrapped with `create-next-app`, using [Prisma](https://www.prisma.io/) as the ORM, and [Tailwind CSS](https://tailwindcss.com/) for styling.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/` - The main application code.
- `prisma/` - Prisma schema and database configuration.
- `public/` - Static assets.

## Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the app for production.
- `npm run start` - Runs the built app in production mode.
- `npm run lint` - Lints the project files.

## Environment Variables

Make sure to create a `.env` file in the root directory (you can use `.env.example` if available) and add your database credentials and any other required environment variables.
